Portal De Seleção De Voluntarios e Lugares.
Minha solução é um portal de cadastro de voluntarios e lugares que estão longe de enchente. Meu objetivo é que seja aberto para o publico. 
Sem login, os cadastros iniciais estão sendo feitos no back-end no front-end coloquei mas a parte da visualização e também poder escolher
algum voluntario ou algum lugar para ficar.
